# pros / cons

## row-oriented

- optimal for read / writes
- compression is not efficient
- aggregation is not efficient
- efficient queries
  - w / multi columns

## column-oriented

- writes are slower
- compress greatly
- amazing for aggregation
- inefficient query
  - w / multi columns
